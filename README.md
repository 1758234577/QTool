QTool 是一个开源 Xposed 模块

## 说明

- 支持Android >= 8.0
- 支持QQ >= 8.7.0

## 功能列表
- 1.Java脚本加载
- 2.语音面板,表情包面板
- 3.QQ辅助功能(消息右下角显示时间,base.apk上传重命名,apk上传失败自动压缩重试,半透明头像上传)
- 4.QQ净化功能(精简表情面板图标,隐藏聊天界面相机图标,隐藏打卡提示)
